Assets Manager

รากฐานที่แข็งแกร่ง (The Intelligent Foundation)

Instant HTML Parser

จบปัญหาการนั่งแกะ Code หน้าเว็บเองทีละบรรทัด เพียงก๊อปปี้ HTML มาวาง ระบบจะสแกนหา Element ทั้งหมดและแปลงเป็นตัวแปร Locator ที่พร้อมใช้งานทันที

สิ่งที่น่าสนใจ:
  • Auto Naming: ตั้งชื่อตัวแปรให้เป็นมาตรฐานเดียวกันทั้งทีม (e.g. LOCATOR_LOGIN_BTN)
  • Framework Support: รองรับ UI ยากๆ เช่น Ant Design หรือ Bootstrap ได้แม่นยำ

Centralized Resource Hub

ศูนย์กลางการจัดการไฟล์ Resource แก้ที่เดียวจบทุกที่ ไม่ต้องตามแก้หลายไฟล์ เปลี่ยนไฟล์ Resource กลางปุ๊บ ทุก Test Case อัปเดตทันที

จุดเด่น:
  • Single Source of Truth: ทีมงานทุกคนใช้ตัวแปรชุดเดียวกัน ลดความซ้ำซ้อน
  • Menu Manager: เครื่องมือพิเศษสำหรับจัดการ XPath เมนูที่มักจะซับซ้อน ให้ง่ายขึ้น

Test Data Management

เครื่องจักรขับเคลื่อนข้อมูล (The Data Engine)

Seamless CSV Workflow

ลืม Excel ไปได้เลย สร้างและแก้ไขข้อมูลทดสอบในโปรแกรม แล้วกดใช้ใน Test Script ได้ทันที ข้อมูลไม่หาย ไม่ต้องย้ายไฟล์ไปมาให้วุ่นวาย

ประโยชน์:
  • Auto-Save Path: ไฟล์จะถูกบันทึกใน resources/datatest ที่ถูกต้องเสมอ
  • Ready to Use: เชื่อมโยงกับปุ่มเลือกข้อมูลในหน้าสร้าง Test Case ทันที

Instant API Builder

แค่มีตัวอย่าง JSON ก็สร้าง Test API ได้ทันที ไม่ต้องเขียนโครงสร้างเอง ระบบจัดการเรื่อง Arguments และ Validation ให้เสร็จสรรพ

ความสามารถ:
  • One-Paste Generation: วาง JSON ปุ๊บ ได้ Keyword ปั๊บ
  • Auto-Validation: แถมโค้ดตรวจสอบ Status Code 200 ให้ในตัว
  • Smart Extract: ดึงค่า Token/ID จาก Response มาใส่ตัวแปรให้ใช้อัตโนมัติ

Keyword Factory

โรงงานประกอบลอจิก (The Logic Assembler)

Visual Logic Builder

สร้าง Logic ซับซ้อนได้เหมือนต่อ LEGO ลาก-วางลำดับการทำงานได้ตามใจชอบ ไม่ต้องกลัว Syntax Error จากการก๊อปวางโค้ดไปมา

ข้อดี:
  • Drag & Drop: ปรับลำดับขั้นตอนได้ง่ายๆ ด้วยเมาส์
  • Legacy Support: ดึง Keyword เก่าที่มีอยู่แล้วมาใช้ซ้ำได้ทันที (Reuse)

Bulk Operation Templates

งานซ้ำๆ จบในคลิกเดียว เช่น กรอกฟอร์มยาวๆ หรือเช็คค่าหลายจุด ระบบเจนโค้ดให้ครบทุกบรรทัดในพริบตา ประหยัดเวลาไปได้มหาศาล

แม่แบบยอดนิยม:
  • Quick Fill: เลือก 20 ช่อง ระบบสร้าง 20 บรรทัดสำหรับกรอกข้อมูลให้ทันที
  • Quick Verify: เลือก Field ที่ต้องการตรวจสอบ ระบบสร้าง Step เช็คค่าให้ครบ

CRUD Generator

สถาปนิกสร้างเทสเคส (The Master Architect)

Standardized Structure

บังคับโครงสร้างมาตรฐาน Setup -> Action -> Verify -> Teardown ให้เป็นนิสัย ทำให้ Test Case ของทุกคนในทีมหน้าตาเหมือนกัน อ่านง่าย และดูแลรักษาง่าย

โครงสร้าง 4 ระยะ:
  • 1. Setup: เตรียมระบบ
  • 2. Actions: ทำรายการ
  • 3. Verify: ตรวจสอบผล
  • 4. Teardown: คืนค่า

Context-Aware Assistance

ปุ่มวิเศษที่รู้ใจ ในช่องกรอกข้อมูล แค่กดปุ่มก็มีรายการ Locator หรือ CSV มาให้เลือก ไม่ต้องจำชื่อตัวแปรแม้แต่ตัวเดียว ลดความผิดพลาดจาก Human Error

ความสามารถ:
  • Smart Select: เลือก Locator จาก List (ดึงจาก Assets อัตโนมัติ)
  • Data Injection: เลือกข้อมูล CSV ระบบสร้าง Syntax ${DATA} ให้เอง
  • Live Preview: เห็นผลลัพธ์โค้ดจริงทันทีที่ขยับเมาส์
